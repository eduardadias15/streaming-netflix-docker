# Plataforma de Streaming Self-Hosted com Docker

Projeto prático desenvolvido na disciplina de Computação em Nuvem na UNICEPLAC.

## Sobre o Projeto
Implementação de uma plataforma de streaming self-hosted inspirada na Netflix,
utilizando apenas ferramentas gratuitas e de código aberto no Debian 12.

## Tecnologias Utilizadas
- **Debian 12** — Sistema operacional do servidor
- **Docker Engine + Docker Compose** — Orquestração dos containers
- **Jellyfin** — Plataforma de streaming de mídia
- **Portainer CE** — Gerenciamento visual dos containers
- **Nginx Proxy Manager** — Proxy reverso e acesso web

## Como Executei
1. Clonei o repositório
2. Executei o comando:
```bash
docker compose up -d
```
3. Acessei os serviços no navegador:
   - 🎥 Jellyfin
   - 🐳 Portainer
   - 🌐 Nginx Proxy Manager
